Hi Camille,

> Do you know when it would be possible to open in ImageJ .lif files
> acquired with the new version of LAS AF (4.0)? (So when Bio-Formats
> will be updated with LAS AF 4?)

This is a question to direct to the OME-users mailing list, which is the
best place to ask Bio-Formats questions.
http://lists.openmicroscopy.org.uk/mailman/listinfo/ome-users

Have you been in contact with the Bio-Formats team previously with an LAS
AF 4 specification document, and/or any sample non-working files? The team
will need them in order to improve the LIF reader. You can upload sample
data using the OME QA system: http://qa.openmicroscopy.org.uk/qa/upload/

Once you report the issue to the OME team, a ticket will created in the OME
issue tracker so that you can follow the status of the new feature.

Regards,
Curtis
