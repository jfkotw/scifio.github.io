Hi Yu,

> But the problem is, the image loading time is so long! It takes almost
> 10 minutes to load a 2 Gigs svs image! Is this normal?

I would suggest trying to load your image in Fiji via the Bio-Formats
Importer plugin:

  http://fiji.sc/Bio-Formats

And see whether the load time is equivalent, or much faster.

If the load time is similar, then it is a limitation of Bio-Formats. In
that case, you can file an issue with the Bio-Formats team:

  http://openmicroscopy.org/info/bio-formats/about/bug-reporting.html

If the load time is much faster, then the problem is with the SCIFIO
ImageIO plugin. In that case, there is already an entry in the issue
tracker for this problem:

  https://github.com/scifio/scifio-imageio/issues/17

Feel free to comment on the issue if you want to be notified of progress.

Regards,
Curtis
