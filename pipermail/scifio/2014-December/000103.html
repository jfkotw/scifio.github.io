Hi Stephan!

>a CellImg that is paged and can be read and written. Is this project
already finished, does this work already?

It is done! The SCIFIOCellImg
<https://github.com/scifio/scifio/blob/dc06e67c8a60ac3e93784722a3dc64d534f24834/src/main/java/io/scif/img/cell/SCIFIOCellImg.java>was
added a few hackathons ago with the guidance of Tobias. There are
heuristics to use it automatically as needed by the ImgOpener
<https://github.com/scifio/scifio/blob/dc06e67c8a60ac3e93784722a3dc64d534f24834/src/main/java/io/scif/img/ImgOpener.java>,
and you can also configure it
<https://github.com/scifio/scifio/blob/dc06e67c8a60ac3e93784722a3dc64d534f24834/src/main/java/io/scif/config/SCIFIOConfig.java#L364>
to be used by default.

If you're interested, there are tickets open to evaluate performance
<https://github.com/scifio/scifio/issues/201>, and supporting large planes
<https://github.com/scifio/scifio/issues/125> via SCIFIOCellImg is a goal
of mine for the next hackathon.

You too! Let me know if you have any questions/problems.
