<div dir="ltr"><div>Hi all,<br><br></div> Yesterday I released <a href="https://github.com/scifio/scifio/releases/tag/scifio-0.5.0" target="_blank">SCIFIO beta 0.5.0</a>. There are a number of new features and significant bug fixes, including:<br>

<ul><li>Use of <a href="https://github.com/imagej/imglib/blob/master/meta/src/main/java/net/imglib2/meta/CalibratedAxis.java" target="_blank">CalibratedAxis</a> in SCIFIO Metadata. This allows for calibration/physical pixel size support without requiring OME-XML.</li>


<li>Updates to use the latest <a href="https://github.com/scijava/scijava-common" target="_blank">Scijava-common</a> (improved logging, Service support, etc..)</li><li>Enhanced <a href="https://github.com/scifio/scifio/blob/master/scifio/src/main/java/io/scif/img/ImgOptions.java">ImgOptions</a> to allow discontinuous <a href="https://github.com/scifio/scifio/blob/master/scifio/src/main/java/io/scif/img/SubRegion.java">sub-region</a> specification</li>

<li>Improved <a href="https://github.com/scifio/scifio/blob/master/scifio/src/main/java/io/scif/img/cell/cache/CacheService.java">CacheService</a> performance/stability</li><li>Fixed memory leaks</li><li>Improved <a href="https://github.com/scifio/scifio/blob/master/scifio/src/main/java/io/scif/filters/Filter.java">Filter</a> functionality</li>

</ul><p> My current goal is to release a <a href="https://github.com/scifio/scifio-bf-compat">SCIFIO-BF-compat</a> update, using SCIFIO 0.5.0 and introducing OME-XML Formats, Metadata and Translators.</p><p> The next SCIFIO release is 0.6.0, which will be a complete refactoring to true N-dimensionality.</p>

<p>- Mark<br></p></div>
