<div dir="ltr"><div><div><div><div><div><div><div>Hi all,<br><br> Today I released <a href="https://github.com/scifio/scifio/releases/tag/scifio-0.7.0">SCIFIO-0.7.0</a>. You can see a full list of the major changes on the <a href="https://github.com/scifio/scifio/issues?milestone=4&state=closed">github issues page</a>.<br>

<br></div>Highlights include:<br></div>* Refactoring to an N-D model from the fixed 5D of Bio-Formats.<br></div>* Upgrading to Java 6.<br></div>* Bug fixes!<br><br></div>Updates to the <a href="https://github.com/scifio/ome-formats">OME-Formats</a> and <a href="https://github.com/scifio/scifio-bf-compat">SCIFIO-BF-compat</a> projects will be coming today/tomorrow. The latest <a href="https://github.com/imagej/imagej">ImageJ2</a> master is already using SCIFIO 0.7.0 and opening N-D images.<br>

<br></div>Development has already begun on <a href="https://github.com/scifio/scifio/issues?page=1&state=open">SCIFIO-0.8.0</a>. The next release will see some polishing of the <a href="https://github.com/scifio/scifio/issues/76">source code</a> and <a href="https://github.com/scifio/scifio/issues/48">documentation</a>, simplification of the <a href="https://github.com/scifio/scifio/issues/74">Metadata API</a>, and a necessary overhaul of <a href="https://github.com/scifio/scifio/issues/69">caching performance</a>.<br>

<br></div><div>Because of the scope of the N-D refactoring, there may still be some issues to address. As always, just let us know if you come across any bugs, or find the N-D API difficult to use.<br><br></div><div>Thanks!<br>
