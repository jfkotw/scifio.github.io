<div dir="ltr"><div>Hi all,<br><br> Just wanted to briefly announce the 1.0.0 release of the <a href="https://github.com/scifio/scifio-bf-compat">SCIFIO-Bio-Formats compatibility layer</a>.<br><br></div> This project is designed to allow Bio-Formats formats to be used by SCIFIO-based applications until these proprietary readers are converted to true SCIFIO formats.<br>

<div><br></div><div> Adding the scifio-bf-compat jar to your classpath should be sufficient to make all the Bio-Formats formats available to SCIFIO.<br><br></div><div> Note that, although SCIFIO is BSD licensed, scifio-bf-compat is GPL, per Bio-Formats itself.<br>

<br></div><div>- Mark<br></div></div>
